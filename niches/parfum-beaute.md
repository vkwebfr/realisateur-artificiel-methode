# Niche : Parfum & Beauté

Prompts et modèles prêts à l'emploi pour les vidéos de parfumerie, cosmétique et beauté de luxe.

---

## Contexte de la niche

Le parfum se vend par l'émotion, jamais par l'explication. On ne filme pas un flacon, on filme ce que le parfum provoque : le désir, la confiance, le souvenir, la tension. Le produit apparaît toujours, mais il n'est jamais seul — il est mis en scène dans un récit sensoriel.

Les codes visuels qui fonctionnent : clair-obscur, lumière rasante dorée, textures tactiles (marbre, cuir, soie, peau), fragments de corps sans visage complet, lenteur des gestes, atmosphère nocturne ou crépusculaire.

Ce qui ne fonctionne pas : éclairage plat de magasin, fond blanc e-commerce, mannequin souriant face caméra, couleurs saturées cheap, ralentis excessifs sans intention.

Marques étalons à étudier : Tom Ford (tension et clair-obscur), Chanel (minimalisme épuré), Le Labo (authenticité brute), Byredo (modernité scandinave), Dior (glamour classique).

## 5 briefs prêts à l'emploi

### Brief 1 — Le lancement mystérieux

Vidéo de 20s pour le lancement d'un nouveau parfum [NOM]. Ambiance : nuit, mystère, élégance. Le flacon apparaît dans 4 à 5 scènes : seul sur une surface sombre, saisi par une main, vaporisé, puis pack-shot final avec logo. On ne voit jamais le visage du personnage. Palette : noir, or, ambre. Référence : Tom Ford Noir. Format : 9:16.

### Brief 2 — Le rituel quotidien

Vidéo de 15s montrant le geste intime d'appliquer [NOM] chaque matin. Ambiance : lumière matinale, sérénité, routine précieuse. Le personnage se prépare face à un miroir, attrape le flacon, vaporise, sourit imperceptiblement. Palette : blanc cassé, beige, lumière dorée chaude. Référence : Glossier, Le Labo. Format : 9:16.

### Brief 3 — Le coffret cadeau

Vidéo de 15s pour une campagne Noël ou Saint-Valentin. Le coffret [NOM] est déballé par des mains soignées. Papier de soie, nœud en satin, flacon révélé. Ambiance feutrée, intime, lumière de bougies. Palette : bordeaux, or, crème. Référence : Cartier holiday campaigns. Format : 9:16.

### Brief 4 — La collection lifestyle

Vidéo de 20s positionnant [NOM] dans un univers de vie. Le flacon est filmé dans différents décors : sur une table de café parisien, dans un sac de voyage en cuir, sur un rebord de fenêtre avec vue urbaine. Pas de personnage. Le parfum raconte un style de vie. Palette : tons neutres, pierre, bois, cuir. Référence : Byredo, Aesop. Format : 16:9.

### Brief 5 — Le duo masculin-féminin

Vidéo de 20s pour un duo [NOM HIM] / [NOM HER]. Deux personnages, jamais visage complet, dans le même espace mais filmés séparément. Leurs gestes se répondent : elle vaporise, il vaporise. Convergence au dernier plan (deux flacons côte à côte). Palette : elle en tons froids (argent, bleu nuit), lui en tons chauds (ambre, noir). Référence : Dior Sauvage x Miss Dior. Format : 9:16.

## Banque de prompts vidéo

5 mouvements vidéo typiques pour les publicités de parfum.

### Mouvement 01 — Reveal lent du flacon

```
Duration: 3s
Use first-frame.png as start frame, last-frame.png as end frame.

Camera: very slow push in (10% over 3 seconds).
Motion: light gradually intensifies on the bottle, revealing 
details of the glass and cap. No other movement.
Lighting: golden rim light slowly brightening.
Style: cinematic, 24fps, film grain, color graded.
```

### Mouvement 02 — Main qui saisit

```
Duration: 3s
Use first-frame.png as start frame, last-frame.png as end frame.

Camera: static.
Motion: hand enters frame slowly from the right, fingers 
close around the bottle with deliberate elegance, lifts 
slightly from surface.
Lighting: consistent throughout.
Style: cinematic, 24fps, film grain, color graded.
```

### Mouvement 03 — Vaporisation slow-motion

```
Duration: 4s
Use first-frame.png as start frame, last-frame.png as end frame.

Camera: static.
Motion: finger presses diffuser, fine mist erupts in slow 
motion, micro-droplets dispersing through golden light, 
settling on skin.
Lighting: rim light catches individual mist particles.
Style: cinematic, 24fps, 50% slow-motion, film grain, color 
graded.
```

### Mouvement 04 — Travelling latéral sur personnage

```
Duration: 5s
Use first-frame.png as start frame, last-frame.png as end frame.

Camera: slow lateral tracking (left to right, 0.3m/s).
Motion: figure walks at natural pace, coat or fabric moving 
subtly, never turning toward camera.
Lighting: transition from warm interior to cool exterior.
Style: cinematic, 24fps, film grain, color graded.
```

### Mouvement 05 — Rack focus vers pack-shot

```
Duration: 4s
Use first-frame.png as start frame, last-frame.png as end frame.

Camera: static, rack focus.
Motion: background element goes out of focus, bottle in 
foreground sharpens into crystal clarity. Logo fades in at 
2-second mark.
Lighting: consistent golden rim light intensifying slightly.
Style: cinematic, 24fps, film grain, color graded.
```

## Pièges courants dans cette niche

- Éclairage plat de type e-commerce : tue l'émotion, rend le flacon banal.
- Fond blanc : acceptable en e-commerce, interdit en publicité cinématographique.
- Visage complet souriant : détruit le mystère. Montrer des fragments.
- Couleurs néon excessives : un reflet subtil sur du sol mouillé, pas une boîte de nuit.
- Ralenti sans motivation : le slow-motion ne remplace pas une intention de mise en scène.
- Logo trop tôt : le logo apparaît au dernier plan, pas avant. Le produit se mérite.
- Musique générique stock : investir dans une piste de qualité ou un compositeur.

## Plateformes et durées recommandées

- **TikTok / Reels** : 15-20s en 9:16 — format roi pour la découverte de marque
- **YouTube Shorts** : 20-30s en 9:16 — permet un plan de plus que Reels
- **Stories Instagram** : 15s max en 9:16 — version condensée (3 plans suffisent)
- **YouTube classique** : 30-60s en 16:9 — version longue avec respiration entre les plans
- **LinkedIn** : 15-20s en 1:1 — pertinent uniquement pour un lancement B2B ou une campagne presse

Publier d'abord sur TikTok/Reels (là où l'algorithme pousse le contenu vertical natif), puis YouTube Shorts 24h après, puis Stories.
