# Niche : Mode & Cosmétique

Prompts et modèles prêts à l'emploi pour les vidéos de marques de mode, collections, lookbooks, cosmétiques, campagnes lifestyle et contenus d'influence.

---

## Contexte de la niche

La mode et la cosmétique ne vendent pas seulement un produit. Elles vendent une identité. Le vêtement, le maquillage ou le soin devient une façon de se tenir, de se montrer, de se reconnaître.

Une bonne vidéo mode-cosmétique doit transmettre une sensation immédiate : confiance, désir, élégance, naturel, audace ou douceur. Le produit doit être visible, mais il doit surtout être porté, vécu, touché, intégré dans un style de vie.

Les codes visuels qui fonctionnent : textures de tissu, peau naturelle, gestes devant un miroir, backstage, détails de couture, marche lente, lumière douce, plans rapprochés, reflets, mouvement des cheveux, matière qui bouge, application produit sur la peau.

Ce qui ne fonctionne pas : mannequins génériques sans personnalité, fonds blancs e-commerce, flat product shots sans mise en scène, poses trop figées, retouches de peau excessives, slogans motivationnels sur écran, esthétique trop stock.

Marques et références à étudier : Sézane pour le naturel parisien, Jacquemus pour la silhouette et la couleur, Glossier pour la peau vivante, Chanel Beauty pour le geste précis, Zara Studio pour le mouvement, Dior Backstage pour l'ambiance coulisses.

## 5 briefs prêts à l'emploi

### Brief 1 — Le lookbook vivant

Vidéo de 20s pour présenter une nouvelle collection de [NOM DE MARQUE]. Ambiance : élégance naturelle, mouvement, confiance. Le modèle traverse 4 à 5 scènes : détail du tissu, ajustement devant miroir, marche dans la rue, gros plan accessoire, silhouette finale. Palette : neutres chauds, lumière naturelle, tons éditoriaux. Référence : Sézane x Vogue Paris. Format : 9:16.

### Brief 2 — Le geste beauté

Vidéo de 15s pour mettre en avant un produit cosmétique [NOM DU PRODUIT]. Ambiance : peau réelle, douceur, intimité. On montre le produit dans la main, la texture, l'application sur la peau, puis le résultat final sans surjouer. La caméra reste proche du visage et des mains. Palette : beige, rose peau, lumière fenêtre. Référence : Glossier. Format : 9:16.

### Brief 3 — Backstage avant défilé

Vidéo de 20s pour raconter les coulisses d'une marque mode ou beauté. Ambiance : tension calme, préparation, précision. On voit les portants, les mains qui ajustent un vêtement, le maquillage, un miroir, puis le modèle qui entre dans la lumière. Palette : noir, crème, flash doux, métal. Référence : Dior Backstage. Format : 9:16.

### Brief 4 — La silhouette signature

Vidéo de 15s centrée sur une pièce forte : robe, veste, sac, lunettes ou rouge à lèvres. Ambiance : désir, attitude, identité. Le produit apparaît par fragments avant la révélation finale : matière, détail, geste, porté, silhouette entière. Palette adaptée à la collection. Référence : Jacquemus. Format : 9:16.

### Brief 5 — La routine premium

Vidéo de 20s pour une routine cosmétique ou mode du matin. Ambiance : calme, soin, élégance quotidienne. Le personnage se prépare dans un intérieur lumineux : peau, texture, vêtement, parfum ou accessoire final. Le résultat doit paraître naturel, pas publicitaire. Palette : lin, lumière dorée, bois clair, blanc cassé. Format : 9:16.

## Pièges courants dans cette niche

* Filmer le produit seul sur fond blanc : trop e-commerce, pas assez désir.
* Utiliser des mannequins trop génériques : il faut une attitude, pas une silhouette vide.
* Lisser la peau de façon excessive : la beauté premium garde une texture réelle.
* Multiplier les poses figées : la mode se vend mieux par le mouvement.
* Montrer trop de logo trop tôt : l'identité doit apparaître après le désir.
* Oublier les détails de matière : tissu, peau, couture et texture créent la valeur.
* Utiliser une lumière trop dure : elle rend les vêtements et la peau moins premium.
* Copier un style runway sans contexte : même un défilé doit raconter une intention.

## Plateformes et durées recommandées

* **TikTok / Reels** : 15-25s en 9:16 — idéal pour lookbook, routine beauté, silhouette ou lancement
* **YouTube Shorts** : 20-30s en 9:16 — utile pour montrer une transformation ou une mini-séquence backstage
* **Stories Instagram** : 15s max en 9:16 — parfait pour teasing, drop de collection ou offre limitée
* **YouTube classique** : 45-90s en 16:9 — pertinent pour campagne de marque, film de collection ou making-of
* **LinkedIn** : 15-30s en 1:1 ou 16:9 — pertinent pour une annonce de marque, collaboration ou lancement presse

Pour une marque mode ou cosmétique, le meilleur point de départ est souvent une vidéo 9:16 de 20 secondes : assez courte pour retenir l'attention, assez longue pour installer une identité.
